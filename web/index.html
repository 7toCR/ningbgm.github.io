NingBGM Demo Page

一个接近 audiogenie.github.io 风格的静态项目页（纯白主题）：支持目录锚点、视频/音频播放器、 并且本版本仅展示仓库 example/ 目录下的 mp4。

Paper Dataset Code example_mp4.json
如何更新展示的视频列表？
编辑 web/data/example_mp4.json，把每个条目的 src 指向 /example/... 下的 mp4 即可。
